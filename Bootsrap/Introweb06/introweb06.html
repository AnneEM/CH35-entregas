Bootstrap Intro



Clases importantes y de mayor uso en bootstrap

A continuación te presento algunas clases importantes para utilizar en bootstrap

Class ="container"

Ocupa las 12 columnas del grid que proporciona CSS, pero deja un margen a los costados

Class="container-fluid"

Ocupa el espacio completo del viewport

Class="row"

Indica que el elemento contenedor se comporta como una fila dentro del grid

Class="col-n/xx"

Dos permiten establecer el npumero de columnas que queremos que ocupe un elemento/componente
Se puede delimitar tanto con nomengraturas predefinidad
Ejemplos= sm, md, lg, xl y xxl.
